在你翻开之前

这本书不会直接"告诉"你答案。它会先问你。

每一个知识点,我都会先抛一个问题,留一个停顿,让你自己先想——然后你点开"揭晓",看你的直觉和市场的逻辑是否对上。这就是苏格拉底式学习:答案不是被灌进去的,是被你自己想出来的。被自己想出来的东西,才忘不掉。

每个概念,我们都会走完四个问题:是什么 → 它的逻辑(为什么有效)→ 好的样子和坏的样子 → 什么时候是高概率。 走完这四步,你不只是"认识"了一个形态,而是真正"读懂"了它背后多空双方在干什么。

先想一想
你打开一张没有任何指标的K线图。在你能"预测"任何东西之前,这张图最基本、最诚实地告诉了你什么?

它告诉你:在每一段时间里,买家和卖家各自把价格推到了哪里,最后谁守住了阵地。仅此而已——但这已经是市场上最真实的信息。没有传言、没有预测,只有"已经发生的搏斗结果"。

价格行为的全部功夫,就是学会从这一连串"搏斗结果"里,读出下一步谁更可能赢。这本书就是教你这个。

第 1 章 · 世界观

1.1 什么是价格行为

是什么

价格行为 (Price Action) 就是:不依赖任何衍生指标,只通过价格本身的走势——每一根K线的形状、它们之间的关系、以及它们组成的结构——来判断市场下一步的倾向。

MACD、RSI、布林带,这些都是把价格"加工"过的二手信息,而且永远滞后。价格行为直接读一手信息:价格本身。Brooks 的整套方法,本质上就是一套"如何读懂裸K线"的语法。

逻辑

为什么要绕开指标、直接读价格?因为所有指标都是价格的函数——它们的信息已经包含在价格里了,而且经过平滑和延迟。当 RSI 告诉你"超买"时,那根强力的趋势棒早就把答案写在图上了。学会读价格,你就站在了信息链的最上游。

连接你的代码

第 1 章 · 世界观

1.2 一根K线是一场战争

是什么

这是 Brooks 世界观的基石,请把它刻进脑子里:每一根K线,都是这段时间里多头和空头一场搏斗的快照。

开盘价 = 开战的位置。收盘价 = 停战时谁占了上风。最高价 = 多头一度打到的最远处。最低价 = 空头一度打到的最远处。影线 = 被对方夺回的失地。

逻辑

一旦你用"战争"的视角看每根棒,图表就活了。一根收在最高点、几乎没有上影线的大阳棒,不是一个抽象的"涨"——它是多头从头压到尾、空头毫无还手之力的一场屠杀。下一根棒,你会本能地预期多头还想继续。

想一想
一根K线,开盘后价格先冲到很高,但收盘时又被打回到接近开盘价,留下一根长长的上影线。这场"战争"发生了什么?谁赢了?

多头先发起进攻、一度大幅推高(那根长上影就是他们打到的最远处)——但在高位,卖方涌入,把价格几乎全部打了回来。战争的结果是:多头的进攻被坚决拒绝了。

这根棒在告诉你:上方有强大的卖压。它本身往往就是一个看跌信号(尤其出现在上涨末端)。你看,我们还没学任何"形态",仅凭"战争视角"就已经读出了第一个反转线索。

第 1 章 · 世界观

1.3 概率,而非预测

是什么

Brooks 从不说"它会涨"。他只说"它大概率涨"。这不是谦虚,是这门手艺的核心纪律:市场上没有确定性,只有概率和盈亏比。

逻辑

任何一个 setup,哪怕是教科书级的完美形态,也可能失败。所以一个成熟交易者真正在算的,是 Brooks 说的交易者方程:

只要 (胜率 × 盈利) 明显大于 (败率 × 亏损),这笔交易就值得做——即使它单笔可能亏。 你的优势不在某一笔,而在你把同一个正期望的 setup 重复一百次。

什么时候高概率

这正是后面所有章节要回答的问题。"高概率"从来不是凭感觉,而是一串可验证的条件叠加:趋势方向、信号棒质量、位置、上下文。我们会一个一个把它们拆开。

你的"趋势强度评分 0–3"和"1腿/2腿测量移动目标",本质就是交易者方程的代码化:强趋势(高胜率倾向)时去够更远的目标(更高盈利),弱趋势时收一点。你早就在算这个方程了,只是现在你知道它叫什么、为什么这么算。
第 2 章 · 你的字母表

2.1 趋势棒与十字星

如果说价格行为是一门语言,那么单根K线就是它的字母。先认字母,才能读句子。

是什么

所有K线,粗分两大类:

趋势棒 (trend bar): 实体大、影线小,方向明确。阳趋势棒 = 这段时间多头主导;阴趋势棒 = 空头主导。
十字星 (doji): 实体很小,开盘和收盘几乎一样。代表多空僵持、犹豫不决。

逻辑

实体的大小,直接等于"赢家赢得有多干脆"。大实体阳棒 = 多头从开盘压到收盘,一路碾压。小实体 doji = 双方拉锯一整段时间,谁也没真正赢。实体讲的是"力量",影线讲的是"被拒绝的尝试"。

想一想
连续出现五六根 doji,挤在一个很窄的价格区间里。这在告诉你市场处于什么状态?这种地方好做吗?

一堆 doji 挤在一起 = 多空力量高度均衡、市场在窄震荡。 Brooks 给这种区域起了个很形象的名字:铁丝网 (barbwire)。

这是最难做的地方之一:顺势者和逆势者都会被反复扫损。所以答案是——不好做,通常应该避开。能"认出该收手的地方",和能认出好机会一样值钱。

bullTrendBar = isBull and bodyRatio >= trendBodyR 就是在用数学定义"趋势棒":实体占整根棒的比例要够大。isDoji 则是比例够小。你代码里那个 0.5 的阈值,就是在画"趋势棒 vs doji"的那条线。
第 2 章 · 你的字母表

2.2 实体、影线与收盘位置

是什么

把一根K线解剖开,有三个关键部位,每个都在讲故事:

实体 (body): 开盘到收盘,代表净胜出的力量。
影线/尾巴 (tail): 价格冲到过、但被夺回的地方,代表被拒绝的方向。
收盘位置 (close): 收在棒的上端、中间还是下端——这是单根棒里最重要的一个信息。

逻辑

为什么收盘位置最重要?因为收盘是这场战争的最终比分。一根棒哪怕一度冲很高,只要收在下端,就说明最后一刻是卖方占了上风。Brooks 看一根棒,眼睛第一个落点永远是:它收在哪里?

好的样子 / 坏的样子

好(强): 阳棒收在最高点附近(上影极短)→ 多头掌控到了最后一刻,毫不松手。
坏(弱): 阳棒虽然是涨的,但收在中间、还拖着长上影 → 多头推上去了,却被卖方按住,这种"涨"是虚的。

第 2 章 · 你的字母表

2.3 强棒与弱棒

是什么

同样是阳棒,有"强"有"弱"。区分它们,是你从"看图新手"迈向"会读棒"的第一道分水岭。

逻辑

左边的强阳棒:大实体、几乎无影线、收在顶部。它说的是"多头全程压制,且到收盘仍在加力"。右边的弱阳棒:小实体、长上影、收在中部。它说的是"多头试图进攻,但上方卖压把价格按了回来"。

两根都是"阳棒",但它们对下一根的暗示截然相反:强棒暗示延续,弱棒暗示犹豫甚至反转。如果你只会区分"红绿",你会把这两根当成一回事——而那正是亏损的开始。

什么时候高概率

强趋势棒出现在趋势方向上时,是最高概率的延续信号。比如上升趋势中,一根收在顶部的强阳棒突破前高——这几乎是教科书级的"多头还要继续"。反过来,弱棒出现在趋势末端,常是动能耗尽的早期预警。

想一想
在一段强劲的上涨里,突然冒出一根很大的阳棒,但它收在最高点、还拉出一根长长的上影线。结合你刚学的——这是好事还是警告?

这是一个警告,而不是好事。在 Brooks 体系里,这种在趋势末端出现的、带长上影的超大棒,很可能是高潮棒 (climax / 衰竭):多头用尽全力做了最后一冲,但上方立刻被卖方迎头压回。

力气在一根棒里全使完了 = 后续没人接力了。这种"看起来很强、其实是强弩之末"的棒,常常标记着趋势的顶部。你看,光会看"实体大不大"还不够,位置(在趋势的哪个阶段)+ 影线 一起,才读出了相反的含义。这就是上下文的威力,我们后面会反复回到它。

第 2 章 · 你的字母表

2.4 信号棒:可以下单的那根

是什么

信号棒 (signal bar): 一根让你据此决定下单的棒。它本身不是入场——它是"如果下一根价格越过我,就进场"的那个触发条件。

逻辑

Brooks 几乎从不"市价追入"。他的标准做法是:看中一根信号棒,然后把买入止损单挂在它的上方一跳(做多)。只有当价格真的向上越过信号棒高点、把你的单子打成交,才说明"市场用真金白银确认了方向",你才进场。

这一跳的越过,就是市场替你做的确认。如果价格连信号棒的高点都够不到,说明买方根本没力气,这笔交易压根不该发生——你的单子自动落空,完美避开。

好的信号棒 / 坏的信号棒

好: 顺着趋势方向的、强实体、收盘有力、位置合理(比如上升趋势回调到EMA后的一根阳棒)。
坏: 又大又重叠的棒(止损太远、盈亏比差),或者出现在铁丝网/震荡里的棒(方向随机)。

第 3 章 · K线之间的对话

3.1 内包棒与外包棒

单根棒是字母,字母之间的关系才组成单词。从这里开始,我们看"两根棒在一起说了什么"。

是什么

内包棒 (inside bar): 当前棒的高低点,完全被前一根棒包住。
外包棒 (outside bar): 当前棒反过来,完全包住了前一根棒(更高的高、更低的低)。

逻辑

内包棒 = 收缩、暂歇、犹豫。 市场的波动突然变小,双方都在观望。它常常是"突破前的蓄力"——像弹簧被压住。
外包棒 = 波动爆发、双方激烈交火。 一根棒里既创了新高又创了新低,说明多空都很猛,但也意味着混乱——外包棒的信息其实常常是矛盾的,要小心。

什么时候高概率

内包棒在强趋势中出现时最有用:它是趋势中一次微小的喘息,顺势方向突破它,往往是低风险的加仓点(止损可以放得很近)。而内包棒出现在震荡区间里则意义不大,因为四周本来就没方向。

第 3 章 · K线之间的对话

3.2 ii 与 ioi

是什么

把内包/外包棒串起来,就有了 Brooks 常用的两个突破前形态:

ii: 连续两根内包棒(inside-inside)。波动连续收缩两次。
ioi: 内包-外包-内包(inside-outside-inside)。先收缩、再爆一下、又收缩。

逻辑

这两个形态的共同点是:能量在被压缩。 连续的收缩意味着多空僵持到了极点,而僵持从来不会永远持续——它必然以一次突破收场。ii / ioi 就像拉满的弓,你不知道箭往哪飞,但你知道它快要飞了。

什么时候高概率

ii / ioi 出现在一个明确趋势的回调末端时,是绝佳的顺势突破点:能量压缩 + 趋势方向 = 高概率向趋势方向爆发,而且止损极近(就放在小形态的另一端),盈亏比非常好。这正是 Brooks 偏爱它们的原因。

想一想
一个 ii 形态出现了,但它正好夹在一片杂乱重叠、毫无方向的铁丝网中央。你该用它做突破吗?

不该。 ii 告诉你"能量在压缩、即将突破",但它没有告诉你突破的方向。在铁丝网里,周围环境本身就是随机的,所以这次突破极可能是"突破即失败"——向上假突破,然后掉头,反之亦然。

这里藏着第 3 章最重要的一课:同一个形态,在不同的上下文里,价值天差地别。 ii 在强趋势回调末端是金矿,在铁丝网里是陷阱。形态本身永远只是半个答案,另一半永远是"它在哪里出现"。

3.3 反转棒

是什么

反转棒 (reversal bar): 一根带有明显反向影线、收盘朝反方向的棒,暗示当前方向可能要变。看涨反转棒通常有长下影 + 收在上端;看跌反转棒则有长上影 + 收在下端。

逻辑

回到"战争"视角:一根看涨反转棒,意味着空头先把价格砸下去(那根长下影),但在低位被多头强力接走、一路买回来收在高点。这场战争里,卖方先赢后输——多头在低位展示了决心。这就是潜在底部的样子。

好的反转棒 / 坏的反转棒

好: 影线明显、收盘强劲、且出现在合理的位置——比如打到了一个重要支撑、或一段下跌的衰竭点。
坏: 孤零零出现在强趋势中途、没有任何支撑配合的反转棒。在强趋势里,单根反转棒被无视、趋势继续碾压,是家常便饭。

什么时候高概率

反转棒绝不单独使用。它的概率高低几乎完全取决于上下文:打在强支撑/阻力上 ✓、是一段趋势的衰竭点 ✓、有更大周期方向支持 ✓。Brooks 甚至更偏爱等"第二次入场 (second entry)"——第一根反转棒失败后的第二次尝试,往往比第一次可靠得多。

第 3 章 · K线之间的对话

3.4 逐棒读盘:把字母连成故事

是什么

逐棒读盘 (bar-by-bar reading) 是 Brooks 的招牌技能,也是这一卷所有内容的汇合点:一根一根地往后读,每根都问"这根棒相对上一根,告诉了我什么?多空的力量天平往哪边动了?"——把零散的字母,连成一个连续的故事。

逻辑

看上面这段。试着用我们学过的东西,把它"讲"出来:

① 一根强阳趋势棒,收在顶部——多头掌控,有人在认真买。
② 一根小内包棒——多头歇了口气,但没有人卖出去,回调很浅。
③ 又一根强阳棒,创了新高并收在高点——多头确认还要继续,买方依然占优。
④ 一根带上影的小阴棒——第一次出现卖方的影子,但很弱,还不足以翻盘。

读完这四根,你不需要任何指标,就已经得出结论:这是一个健康的、多头掌控的上升趋势,目前还看不到反转的真凭实据。 这就是逐棒读盘的力量——它不是预测未来,而是持续地、诚实地描述当下谁在控盘。

综合练习
接着上面那段:第⑤根,出现了一根又大又长、带长上影、收在下端的阴棒,一口气吞掉了前两根阳棒的涨幅。故事现在变成什么了?

故事转折了。这根大阴棒说的是:空头第一次真正、大规模地出手,而且赢得干脆——它不仅强(大实体、收在下端),还抹掉了之前两根棒多头的努力。这是空头力量的第一次实质性宣告。

但请注意你刚学的纪律:这是"警告",还不是"反转确认"。 一根棒掀不翻一个趋势。Brooks 会说:现在要开始留意了,等一个真正的反转结构(比如我们下一卷要讲的趋势线突破 + 测试)来确认。会读棒的人,在反转真正发生前就已经嗅到了味道——但不会因为一根棒就鲁莽反手。

通向"趋势"的门

恭喜——你已经走完了地基。回头看,你现在能做一件你翻开这本书时还做不到的事:对着一张裸K线图,逐根读出多空力量的此消彼长。

我们建立了三件东西,它们会支撑你之后学的一切:

① 一种视角——每根棒都是一场战争,收盘是比分。
② 一套字母表——趋势棒、doji、强棒、弱棒、信号棒。
③ 一条铁律——形态只是半个答案,上下文是另一半;概率,而非预测。

第二卷《趋势》会把这些字母组成第一个完整的"句子":市场是怎么形成趋势的、回调长什么样、H1/H2 怎么数、EMA 缺口为什么是力量的标志、always-in 方向如何翻转——也就是你策略代码里那些逻辑真正的来历。学完第二卷,你再回头看自己写的那套趋势跟随策略,会有一种"原来每一行都对应着一个我亲手想通的道理"的踏实感。

通向下一卷的问题
如果一个"趋势"无非就是一连串"多头持续赢得每一场小战争"——那么,趋势第一次"开始失效"的最早信号,会是哪一根棒身上的什么变化?

最早的信号,往往是"赢得不再干脆":顺势的趋势棒开始变小、开始带反向影线、开始收得不那么靠近极端;或者一次回调,第一次跌破了之前回调都守住的位置(比如 EMA)。力量不是一夜消失的,它是一根棒一根棒地泄气的。

而"如何系统地识别这种泄气、并把它变成可交易的回调与反转 setup"——正是第二卷的全部内容。带着这个问题去学,你会学得又快又深。
