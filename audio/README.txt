读棒 · 沉浸阅读音频
==================

把你自己的背景音(白噪音 / 海浪 / 雨 / 森林 / 火 等)放进这个文件夹,就能在书页
右下角 🎧 面板里选择、循环播放。

怎么加自己的音频
----------------
1. 把音频文件复制进这个 audio/ 文件夹。建议用 .mp3(体积小、所有浏览器都支持)。
2. 打开 sounds.json,在 "sounds" 数组里为每个文件加一行:
       { "label": "按钮上显示的名字", "file": "你的文件名.mp3" }
   - label 是面板按钮上显示的字(随便取,如「海浪」「咖啡馆」)。
   - file 要和实际文件名完全一致(含大小写和扩展名)。
3. 刷新书页 → 🎧 面板里就会出现对应按钮,点一下循环播放。

当前 sounds.json 里已经预留了 ocean.mp3 / rain.mp3 / forest.mp3 / fire.mp3 四个名字。
你可以:把同名文件放进来,或直接改 sounds.json 用你自己的文件名。

注意
----
- 必须用本地服务器或 GitHub Pages 打开(file:// 直接双击会被浏览器拦截,读不到 json/音频)。
- 这些音频会随仓库一起提交并部署到 GitHub Pages,注意体积:每个文件建议 < 1~2 MB
  (循环音几十秒就够;太大的会拖慢加载、撑大仓库)。
- 音量用 🎧 面板里的滑块调;选「关」停止。
- 去哪找免费可商用的音频:搜索 "CC0 / public domain ambience"(如 freesound.org 里
  CC0 许可的、Pixabay 的免费音效)。注意版权,别用受版权保护的音乐。
